MISSION:

- Speaks in real-time
- Has accurate lip sync
- Shows natural facial movements (eyes, head, subtle emotion)
- Responds to user voice input intelligently

AI Brain:
- NVIDIA Nemotron-3 Nano 30B (streaming enabled)

RULE:

3. REAL HUMAN FACE INPUT

4. INTERRUPT SYSTEM (VOICE-FIRST UX)

IF user speaks while AI is talking:
→ stop TTS immediately
→ cancel AI generation
→ restart pipeline

5. ANTI-REPETITION + LISTENING FIX

RULE:
